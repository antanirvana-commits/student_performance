# Tableau Dashboard Blueprint

## Recommended dashboard sheets
1. KPI cards: total students, average exam score, percentage high performance
2. Scatter plot: Hours Studied vs Exam Score
3. Scatter plot: Attendance vs Exam Score
4. Heatmap: correlation between numeric factors
5. Bar chart: performance category distribution
6. Bar chart: average exam score by parental involvement or access to resources

## Dashboard layout
- Top row: 3 KPI cards
- Middle row: 2 scatter plots
- Bottom row: class distribution and average score by category
- Right panel: filters for gender, school type, access to resources, motivation level

## Tableau connection steps
1. Open Tableau Desktop or Tableau Public
2. Connect to `student_performance_tableau_source.csv`
3. Create calculated fields from `calculated_fields.txt`
4. Build the suggested sheets
5. Publish to Tableau Public or export workbook file
